JAVASCRIPT_定义文本动作

  1. 外部引用
    2. <script src="test.js"></script>
  2. strict模式：强制要求使用var定义变量
  3. 字符串：以单引号'或双引号"括起来的任意文本
    • length;字符串长度
    • toUpperCase();字符串大写
    • toLowerCase();字符串小写
    • indexOf();返回指定字符串出现的位置
    • substring();指定索引区间的子串
  4. 数组：一组按顺序排列的集合，集合的每个值称为元素
    • length;数组长度
    • indexOf();返回指定元素出现的位置
    • slice;返回指定元素出现的位置
    • push();末尾添加若干元素
    • pop();删除最后一个元素
    • unshift();开头添加若干元素
    • shift();删除第一个元素
    • sort();元素排序
    • reverse();元素反转
    • splice();指定的索引区间删除若干元素，然后再从该位置添加若干元素
    • concat();连接数组
    • join();指定字符串连接数组中的元素
  5. 对象：是一组键值对的结构
    • delete;删除对象的一个属性
  6. map:更高级的对象，具有极快的查找速度
    • var x=new Map();定义空map
    • x.set('wowo',22);添加键值对
    • x.get('wowo');查看键值对
    • x.has('wowo');检查是否存在
    • x.delete('wowo');删除键值对
  7. set;没有重复元素的数组
    • var x=new Set();定义空set
    • x.add(x);添加元素
    • x.delete(x);删除元素
