  13. 高阶函数
    • map函数：把一个function作用到Array的每一个元素上
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.map(String)
                            console.log(new_arr)
                            // Array(4) ["1", "2", "3", "4"]
    • reduce函数：把一个function作用到Array的每一个元素上; 这个function必须接收两个参数，并把结果继续和序列的下一个 元素做累计计算。
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.reduce(function (x,y){
                                return x+y
                            })
                            console.log(new_arr)
                            // 10
    • filter函数：把一个function作用到Array的每一个元素上; 根据返回值是true或false决定是否保留元素。
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.filter(function(x){
                                return x % 2 !==0
                            })
                            console.log(new_arr)
                            // Array(2) [1, 3]
    • sort排序(直接排序会按ASCII进行比较)
      •                             var arr = [2,3,4,1];
                            var new_arr = arr.sort(function(x,y){
                                if (x < y){
                                    return -1
                                }
                                if (x > y){
                                    return 1
                                }
                                return 0
                            })
                            console.log(new_arr)
                            // Array(4) [1, 2, 3, 4]
    • every()：判断数组的所有元素是否满足测试条件
      •                             var arr = [2,3,4,1];
                            var new_arr = arr.every(function(x){
                                return x > 2
                            })
                            console.log(new_arr)
                            // false
    • find()：查找符合条件的第一个元素，找不到就返回undefined
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.find(function(x){
                                return x == 0
                            })
                            console.log(new_arr)
                            // undefined
    • findIndex()：查找符合条件的第一个元素，找到就返回该元素的索引
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.findIndex(function(x){
                                return x == 3
                            })
                            console.log(new_arr)
                            // 2
    • 闭包：函数作为返回值
      •                             function lazy_sum(arr){
                                var sum = function(){

      1. navigator.appName;浏览器名称
      2. navigator.appVersion;浏览器脚本
      3. navigator.language;浏览器设置的语言
      4. navigator.platform;操作系统类型
      5. navigator.userAgent;浏览器设定的User-Agent字符串

      1. screen.width;屏幕宽度
      2. screen.height;屏幕高度

      1. location.protocol;当前URL的协议
      2. location.host;当前URL的主机名
      3. location.port;当前URL的端口
      4. location.pathname;当前URL的路径
      5. location.search;当前URL的参数
      6. location.hash;当前URL的锚

      1. document.getElementById();用id定位DOM树

  18. DOM方法

    • 创建DOM
      1. document.createElement();创建元素节点
      2. document.createAttribute();创建属性节点
      3. document.createTextNode();创建文本节点
      4. document.appendChild();把新的节点添加到指定节点
    • 插入DOM
      1. parent_node.insertBefore(create_node,reference_node);把子节点插入到指定位置
    • 删除DOM
      1. parent_node.removeChild(child_node);调用父元素的removeChild()即可删除子元素
  19. dom属性
    • node.innerHTML;更新整个标签
    • node.innerText;更新文本内容
    • node.nodeName;规定节点名称
      1. 元素节点的 nodeName 与标签名相同
      2. 属性节点的 nodeName 与属性名相同
      3. 文本节点的 nodeName 始终是 #text
      4. 文档节点的 nodeName 始终是 #document
    • node.nodeValue;规定节点值
      1. 元素节点的 nodeValue 是 undefined 或 null
      2. 文本节点的 nodeValue 是文本本身
      3. 属性节点的 nodeValue 是属性值
    • node.nodeType;返回节点的类型
      1. 元素：1
      2. 属性：2
      3. 文本：3
      4. 注释：8
      5. 文档：9

  21. jQuery
    • 选择器
      • 元素选择器
        1. $('tagname');选取所有包含该标签名的标签
        2. $('selector');选取所有包含该选择器的标签
        3. $('tagname selector');选取所有包含该标签名且包含该选择器的标签
      • 属性选择器
        1. $('[element_name]');选取所有包含该属性名的标签
    • 遍历
      • 祖先
        1. $(selector).parent();返回被选元素的直接父元素
        2. $(selector).parents();返回被选元素的所有祖先元素
        3. $(selector).parentsUntil($(selector));返回介于两个元素之间的所有祖先元素
      • 后代
        1. $(selector).children();返回被选元素的所有直接子元素
        2. $(selector).find('*');返回被选元素的所有后代元素
      • 同胞
        1. $(selector).siblings();返回被选元素的所有同胞元素
        2. $(selector).next();返回被选元素的下一个同胞元素
        3. $(selector).nextAll();返回被选元素的所有跟随同胞元素
        4. $(selector).nextUntil($(selector));返回介于两个元素之间的所有跟随同胞元素
      • 过滤
        1. $(selector).first();返回被选元素的首个元素
        2. $(selector).last();返回被选元素的首个元素
        3. $(selector).eq(index);返回被选元素中指定索引的元素
        4. $(selector).filter($(selector));返回被选元素中指定筛选条件的元素
        5. $(selector).not($(selector));与filter()相反
    • 效果
      • 隐藏/显示
        1. $(selector).hide()隐藏标签(不会改变DOM树的结构)
        2. $(selector).show()显示标签
        3. $(selector).toggle();即可显示又可隐藏
      • 淡入淡出
        1. $(selector).fadeIn(speed,function);淡入已隐藏的元素
        2. $(selector).fadeOut(speed,function);淡出可见的元素
        3. $(selector).fadeToggle(speed,function);既可淡出又可淡入
        4. $(selector).fadeTo(speed,function);允许渐变为给定的不透明度
      • 滑动
        1. $(selector).slideDown(speed,function);滑出已隐藏的元素
        2. $(selector).slideUp(speed,function);滑入可见的元素
        3. $(selector).slideToggle(speed,function);既可滑出又可滑入
      • 动画
        1. ${selector}.animate({css},speed,function);添加css样式的动画效果
      • 停止滑动和动画
        1. ${selector}.stop(stopAll,goToEnd);停止正在运行中的滑动和动画
        2. 可选的 stopAll 参数规定是否应该清除动画队列。默认是 false，即仅停止活动的动画，允许任何排入队列的动画向后执行。

    • 操作DOM
      • 获取和设置
        1. ${selector}.text();设置或返回所选元素的文本内容
        2. ${selector}.html();设置或返回所选元素的内容(包括HTML标记)
