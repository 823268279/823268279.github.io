  13. 高阶函数
    • map函数：把一个function作用到Array的每一个元素上
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.map(String)
                            console.log(new_arr)
                            // Array(4) ["1", "2", "3", "4"]
    • reduce函数：把一个function作用到Array的每一个元素上; 这个function必须接收两个参数，并把结果继续和序列的下一个 元素做累计计算。
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.reduce(function (x,y){
                                return x+y
                            })
                            console.log(new_arr)
                            // 10
    • filter函数：把一个function作用到Array的每一个元素上; 根据返回值是true或false决定是否保留元素。
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.filter(function(x){
                                return x % 2 !==0
                            })
                            console.log(new_arr)
                            // Array(2) [1, 3]
    • sort排序(直接排序会按ASCII进行比较)
      •                             var arr = [2,3,4,1];
                            var new_arr = arr.sort(function(x,y){
                                if (x < y){
                                    return -1
                                }
                                if (x > y){
                                    return 1
                                }
                                return 0
                            })
                            console.log(new_arr)
                            // Array(4) [1, 2, 3, 4]
    • every()：判断数组的所有元素是否满足测试条件
      •                             var arr = [2,3,4,1];
                            var new_arr = arr.every(function(x){
                                return x > 2
                            })
                            console.log(new_arr)
                            // false
    • find()：查找符合条件的第一个元素，找不到就返回undefined
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.find(function(x){
                                return x == 0
                            })
                            console.log(new_arr)
                            // undefined
    • findIndex()：查找符合条件的第一个元素，找到就返回该元素的索引
      •                             var arr = [1,2,3,4];
                            var new_arr = arr.findIndex(function(x){
                                return x == 3
                            })
                            console.log(new_arr)
                            // 2
    • 闭包：函数作为返回值
      •                             function lazy_sum(arr){
                                var sum = function(){
                                    return arr.reduce(function(x,y){
                                        return x + y
                                    })
                                }
                                return sum
                            }
                            var f = lazy_sum([1,2,3,4])
                            console.log(f)
                            // function(){...}
                            console.log(f(x))
                            // 10

  17. 浏览器对象
    • window对象:全局作用域，表示浏览器窗口
      1. window.innerWidth;获取浏览器窗口内部宽度
      2. window.innerHeight;获取浏览器窗口的内部高度
      3. window.outerWidth;获取浏览器窗口的整个高度
      4. window.outerHeight;获取浏览器窗口的整个宽度
    • navigator对象：表示浏览器内核
      1. navigator.appName;浏览器名称
      2. navigator.appVersion;浏览器脚本
      3. navigator.language;浏览器设置的语言
      4. navigator.platform;操作系统类型
      5. navigator.userAgent;浏览器设定的User-Agent字符串
    • screen对象：表示浏览器屏幕
      1. screen.width;屏幕宽度
      2. screen.height;屏幕高度
      3. screen.colorDepth;颜色位数
    • location对象：表示当前URL
      1. location.protocol;当前URL的协议
      2. location.host;当前URL的主机名
      3. location.port;当前URL的端口
      4. location.pathname;当前URL的路径
      5. location.search;当前URL的参数
      6. location.hash;当前URL的锚
      7. location.href;当前完整url
    • document对象：表示当前DOM树的根节点
      1. document.getElementById();用id定位DOM树
      2. document.getElementsByClassName;用class定位DOM树
      3. document.getElementsByTagName;用标签名定位DOM树
      4. document.cookie;获取cookie
