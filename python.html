python脚本语言

  1. 字符串：一组有序，不可变的，可重复的数据
    • ord(x);把字符编码成整数
    • chr(x);把编码转换成对应的字符
    • encode(x);把str编码成bytes
    • decode(x)把bytes解码成str
    • len(x);长度
    • split();把字符串转换为列表
    • join();把列表转换为字符串
    • %s;字符串替换
    • replace(x);字符串替换
    • %d;整数替换
    • %f;浮点数替换
    • %x;十六进制整数替换
  2. 列表：一组有序，可变的，可重复的数据
    • [x];索引
    • [x:y];切片
      1. [:x];前x个元素
      2. [-x:];后x个元素
      3. [::x];每隔x取一个元素
    • append(x);末尾插入
    • len(x);长度
    • pop(x);末尾删除
    • insert(num,x);指定插入
    • pop(num);指定删除
  3. 元组：一组有序，不可变，可重复的数据
    • [x];索引
    • [x;y];切片
    • len(x);长度
  4. 字典:一组无序，可变的，可重复的键值对数据
    • get('key');检查是否存在
    • pop('key');删除字典
    • items();以列表返回可遍历的(键, 值)元组数组
  5. 集合：一组无序，不重复的数据
    • x=set();空集合
    • add(x);插入元素
    • len(x);长度
    • update(x);插入列表、元组、字典
    • remove(x);删除元素，没有元素就报错
    • discard(x);删除元素，没有也不会报错
    • s1&s2;集合交集
    • s1|s2;集合并集
    • s1-s2;集合差集
    • pop();随机删除元素
    • clear();清空元素

  8. 函数
    • 函数定义
      •                             def test():
                                x='a'
                                print(x)
                            test()
    • return;返回函数结果
      •                             def test():
                                x='a'
                                return x
                            test()
    • 位置参数
      •                             def test(x):
                                print(x)
                            test(2)
    • 默认值参数
      •                             def test(x,y=3):
                                print(x)
                                print(y)
                            test(2)
    • 可变参数
      •                             def test(*x):
                                for i in x:
                                    print(i)
                            list=[1,2,3,4]
                            test(*list)
    • 关键字参数
      •                             def test(**x):
                                for i,n in x.items():
                                    print(i,n)
                            dict={'name':'wowo','age':'22'}
                            test(**dict)
    • 递归函数
      •                             def test(x):
                                    if x==1:
                                            return 1
                                    return x*test(x-1)
                            print(test(3))

      1. print();打印
      2. input();输入
      3. abs(x);返回绝对值
      4. max(x,y,z);返回最大值
      5. hex(x);整数转换为16进制的字符串
      6. isinstance();检查数据类型
      7. lower();转换为小写
      8. sum();求和
