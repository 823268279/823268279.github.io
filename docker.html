docker容器

  1. 登录docker hub
    2. docker login;
  2. 登出docker hub
    3. docker logout;
  3. 启动docker守护进程
  4. docker search docker_name;搜索docker镜像
    5. service docker start;
  5. 获取docker镜像
    6. docker pull docker_name;
  6. 推送本地镜像到docker hub(登录状态)
    7. docker push docker_name;
  7. 启动容器并进入终端
    8. docker run -it docker_name /bin/bash;
    • -i;表示交互式操作
    • -t;表示终端
    • -d;表示让容器在后台运行
    • -p;表示将容器内部使用的网络端口映射到宿主机
    • /bin/bash;交互式shell
  8. 进入后台运行中的容器
    9. docker exec -it docker_id /bin/bash;
  9. 停止运行容器
    10. docker stop docker_id;
  10. 重启被停止的容器
    11. docker restart docker_id;
  11. 退出容器终端
    12. exit;
  12. 查看运行中的docker日志
    13. docker logs -f docker_id;
  13. 查看容器内部运行的进程
    14. docker top docker_id;
  14. 查看容器底层信息
    15. docker inspect docker_id;
  15. 查看运行中的容器
    16. docker ps;
  16. 删除容器
    17. docker rm -f docker_id;
  17. 导出容器
    18. docker export docker_id xxx.tar;
  18. 导出容器
    19. cat docker/xxx.tar | docker import REPOSITORY:TA;
  19. 查看本地所有镜像
    20. docker images;
  20. 修改容器之后创建成新容器
    21. docker commit -m='description' -a='auther' docker_id new_docker_name:tag;
