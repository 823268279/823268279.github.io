GIT版本控制器

  • 本地库关联远程库
    • git remote add origin https;
  • 拉取远程库
    • git clone https;
  • 查看仓库状态
    • git status;
  • 添加所有修改到缓存区
    • git add --all;
  • 将缓存区提交到本地仓库
    • git commit -m 'description';
  • 查看分支
    • git branch;
  • 创建分支
    • git branch branch_name
  • 切换分支
    • git switch branch_name
  • 合并某分支到当前分支
    • git merge branch_name
  • 删除分支
    • git branch -d branch_name
  • 查看日志
    • git log
