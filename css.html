CSS_定义文本样式

  1. 外部引用
    2. <link rel="stylesheet" href="url">
  2. 内部引用
    3. <style> div{background-color:red} </style>
  3. 内联样式
    4. <p style="background-color:red; width: 100px;"></p>
  4. 选择器
    • id选择器(用“#”表示，再用id进行选择)
      •                             #{background:red;}
    • class选择器(用“.”表示，再用class进行选择)
      •                             .{background:red;}
    • 派生选择器
      •                             li strong{background:red;}
    • 群组选择器(用标签选择)
    • 通配选择器(用“*”表示，作用全局)
      •                             *{background:red;}
    • 层次选择器(按标签层选择)
    • 属性选择器(按标签中的属性进行选择)
      •                             input[type="text"]{background:red;}
    • 伪类选择器(用于向某些标签添加特殊效果)
      1. 一般伪类选择器
        2.                                     a:link{background:red;} 访问前渲染(只能给a标签)
                                    a:visited{background:green;}；访问后渲染(只能给a标签)
                                    a:hover{background:violet;}；鼠标移入时才会渲染(适用所有标签)
                                    a:active{background:black;}；鼠标按下时才会渲染(适用所有标签)
                                    div:after{content:"text"}；在元素文本之后添加text
                                    div:before{content:'text'}；在元素文本之前添加text
                                    :checked{width:100px;height:100px;} 只渲染含有checked属性的元素(checkbox)
                                    :focus{background-color:red;} 只渲染含有focus属性的元素(input)
      2. 结构伪类选择器
        3.                                     li:nth-of-type(x){background-color:red;}；表示按类型选择渲染，x从1开始(不受其他标签影响)
                                    li:nth-child(x){background-color:red;}；表示按类型选择渲染，x从1开始(受其他标签影响)
                                    li:first-of-type{background-color:red;}；只渲染该type的第一个标签
                                    li:last-of-type{background-color:blue;}；只渲染该type的最后一个标签
                                    li:only-of-type{background-color:green;}；只渲染该type只有一个标签时
  5. 背景样式
    • width:100px;定义标签宽度
    • height:100px;定义标签高度
    • background-color:red;定义标签背景颜色
    • background-image:x.png;定义标签背景图片
    • background-repeat:x;定义背景图片的平铺方式
      1. repeat-x;x轴平铺
      2. repeat-y;y轴平铺
      3. repeat:x,y;x,y都进行平铺(默认值)
      4. no-repeat;都不平铺
    • background-position:x y;定义背景图片的位置
      1. x,y;可以用像素表示(px)
      2. x,y;可以用位置表示(left、center、right;top、center、bottom)
    • background-attachment:x;背景图片随滚动条的移动方式
      1. scroll;背景位置是按当前元素进行偏移()
      2. fixed;背景位置是按照浏览器进行偏移
  6. 边框样式
    • border-style:x;边框线的样式
      1. solid;实线
      2. dashed;虚线
      3. dotted;点线
    • border-width:10px;定义边框线的粗细
    • border-color:red;定义边框线的颜色
    • 单独定义top、bottom、left、right的边框
      1. border-top-style:solid;定义上边框线的样式
      2. border-bottom-width:3px;定义下边框线的粗细
      3. border-left-color：red;定义左边框线的颜色
    • border-radius:10px;定义边框圆角
  7. 文字样式
    • font-family:x;定义字体类型
      1. simsun;宋体(衬线体)
      2. microsoft yahei;微软雅黑(非衬线体)
    • font-size:x;定义字体大小
      1. 像素定义，像素越大字体越大
      2. medium;正常大小
      3. large;大
      4. x-large;较大
      5. xx-large;最大
      6. small;小
      7. x-small;较小
      8. xx-small;最小
    • font-weight:x;定义字体粗细
      1. 数值定义，数值越大，字体越粗
      2. normal;正常粗细(默认值)
      3. bolder;字体加粗
    • font-style:x;定义字体样式
      1. normal;正体(默认)
      2. italic:斜体
    • color:red;定义字体颜色
  8. 文本样式
    • text-decoration:x;文本修饰
      1. underline;定义文本下划线
      2. line-through;定义文本删除线
      3. overline;定义文本上划线
      4. none;表示无样式
    • text-transform:x;英文修饰
      1. lowercase;英文小写
      2. uppercase;英文大写
    • text-indent:x;文本缩进
      1. px表示固定首行缩进
      2. em表示相对首行缩进(相对字体大小而变化)
    • letter-spacing:x;字间距
      1. px表示固定字间距
      2. em表示相对字间距(相对字体大小而变化)
    • word-spacing:x;词间距
      1. px表示固定词间距
      2. em表示相对词间距(相对字体大小而变化)
    • text-align:x;文本对齐方式
      1. left;文本左对齐
      2. center;文本居中
      3. right;文本右对齐
      4. justify;两端对齐
    • line-height:x;文本行高(上边距+字体大小+下边距=行高)
      1. 像素定义行高
      2. 数值定义行高(字体大小*数值=行高)
    • 文本字间距
      1. letter-spacing:10px;中文字间距
      2. word-spacing:10px;英文词间距
    • 英文词折行
      1. word-break:break-all;强制折行
      2. word-wrap:break-word;非强制折行
  9. 伪类
    • :lick{style};向未被访问的链接添加样式
    • :visited{style};向已被访问的链接添加样式
    • :hover{style};当鼠标悬浮在元素上时，向元素添加样式
    • :active:{style};向被激活的元素添加样式
    • :focus;向拥有键盘输入焦点的元素添加样式
    • :first-child{};向元素的第一个子元素添加样式
    • :lang{};向带有指定lang属性的元素添加样式
    • :first-letter;向文本的第一个字母添加特殊样式
    • :first-line;向文本的首行添加特殊样式
    • :before;在元素之前添加内容
    • :after;在元素之后添加内容
  11. 列表样式
    • list-style-image:url('test.png');自定义列表项目符号
  12. 轮廓样式
    • outline-color:red;定义轮廓颜色
    • outline-style:x;定义轮廓样式
      1. none;无轮廓(默认值)
      2. dotted;点状轮廓
      3. dashed;虚线轮廓
      4. solid;实线轮廓
      5. double;双线轮廓
      6. groove;3D凹槽轮廓
      7. ridge;3D凸槽轮廓
      8. inset;3D凹边轮廓
      9. outset;3D凸边轮廓
    • outline-width:x;定义轮廓宽度
      1. thin;细轮廓
      2. medium;中等轮廓(默认)
      3. thick;粗轮廓
      4. length;自定义
  13. 复合样式：一个css属性控制多种样式
    • 背景中的样式没有顺序
    • 边框中的样式没有顺序
    • 文本中的属性有顺序(size和family必须放在最后)
    • 轮廓中的属性按颜色、样式、宽度排序
  14. 盒子模型
    • content:内容区域，由width和height决定
    • padding：内边距，由像素控制距离，不能为负数
    • border：包装盒，占10个固定像素
    • margin：外边距，由像素控制距离，可以为负数
      1. 盒子之间同时存在上下margin，margin距离会叠加(叠加值取最大值)；只用一个margin即可解决
      2. 嵌套时，margin-top会影响到父元素；用padding替代父容器进行解决
    • box-sizing:border-box;盒模型自适应
  15. 标签分类：布局一般用块标签，修饰文本一般用内联标签
    • block(块)：div、p、ul、li、h......
      1. 独占一行
      2. 支持所有样式
      3. 不写宽度的时候，宽度由父容器决定
      4. 所占区域是一个矩形
      5. Inline(内联)：span、a、em、strong、img......
        1. 挨在一排，排列显示
        2. 有些样式不支持：width、height、margin、padding
        3. 不写宽度的时候，宽度由内容决定
        4. 所占区域不一定是矩形
        5. 内联之间会有间隙(换行产生的)
      6. display:block;将标签的特定属性强制改成一个块的特定属性
      7. display:inline;将标签的特定属性强制改成一个内联的特定属性
      8. display:inline-block;将标签的特定属性强制改成一个内联-块的特定属性
    • 溢出隐藏
      • overflow:visible;默认值，内容不会被剪切，会呈现在元素框之外
      • overflow:hidden;用剪切的方式隐藏溢出字体
      • overflow:scroll;用滑动条的方式隐藏字体(没有溢出也会显示滑动条)
      • overflow:auto;用滑动条的方式隐藏字体(只有溢出才会加滑动条)
      • overflow:inherit;从父元素继承overflow属性的值
    • 透明度与手势
      • opacity:0.5;所有子标签透明
      • background-rgba(255,0,0,0.3);只有背景色会透明
      • cursor:x;手势
        1. cursor:default;鼠标触碰变箭头图标
        2. cursor:pointer;鼠标触碰变小手图片
        3. cursor:move;鼠标触碰变移动图标
        4. cursor:help;鼠标触碰变帮助图标
        5. cursor:url(./鼠标.ico),auto;自定义鼠标触碰图标
    • float浮动
      • 定义：脱离文档流，延迟父容器靠左或靠右进行排列
      • 注意点
        1. 只会影响到后面的元素
        2. 元素的内容默认提升半层
        3. 默认宽根据内容决定
        4. 根据父容器换行排列
      • 清除浮动：清除之后不受浮动元素影响
        1. clear left:清除左浮动
        2. clear right:清除右浮动
        3. clear both:清除左右浮动
    • position:x;定位
      • static;静态定位(默认值)
        • 文档流，忽略定位
      • adsolute;绝对定位
        • 可以使块元素具有内联特性
        可以是内联元素具有块特性
        脱离文档流，根据定位显示，依据父级元素为参照物
      • relative;相对定位
        • 文档流，根据定位显示
      • fixed;固定定位
        • 脱离文档流，根据定位显示，依据浏览器窗口为参照物
      • z-index；调整定位层级
